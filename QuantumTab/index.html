Quantum Tab Privacy Policy

What Data We Collect

Quantum Tab is designed with privacy as a core principle. Our extension operates locally on your device and collects minimal data to provide you with an enhanced new tab experience:

  • Website Visit Counts: URLs and visit frequencies for websites you visit (stored locally)
  • Widget Configurations: Your personalized dashboard layout and widget settings
  • Background Images: Custom background images you upload or generate
  • Quick Action Buttons: Custom shortcuts and bookmarks you create
  • Language Preferences: Your selected interface language (English or Afrikaans)
  • GitHub Repository Data: Repository information when you connect via Personal Access Token
Important: No personal identifying information, browsing history, or sensitive data is collected. All data remains on your local device.

How We Use Your Data

The data collected by Quantum Tab is used exclusively to enhance your productivity and personalize your new tab experience:

  • Website Counter Widget: Display visit statistics and website favicons
  • Dashboard Personalization: Remember your widget layout and preferences
  • Background Customization: Store and display your chosen background images
  • Quick Access: Provide instant access to your frequently used websites
  • GitHub Integration: Show pull request information from your repositories
  • AI Background Generation: When enabled, communicate with your chosen LLM provider

Where Is Your Data Stored?

Quantum Tab prioritizes data privacy by storing all information locally on your device using Chrome's secure storage API. Your data never leaves your browser unless you explicitly enable external integrations:

Local Storage (Chrome Extension Storage API)

  • Widget configurations and layouts
  • Website visit counters and statistics
  • Custom background images
  • Quick action button configurations
  • Language preferences
  • Dashboard lock states

Optional External Integrations

When you choose to enable external integrations, limited data may be transmitted to third-party services under your explicit control:

  • GitHub API: Repository data fetched using your Personal Access Token
  • LLM Providers: AI image generation prompts sent to OpenAI, Google Gemini, or Azure OpenAI (only when AI backgrounds are enabled)
Your Control: You can disable any external integration at any time. API keys and tokens are encrypted before local storage.

Data Security

Quantum Tab implements multiple security measures to protect your data:

  • Local-First Architecture: Data remains on your device by default
  • Encrypted API Keys: All third-party API keys are encrypted before storage
  • Secure Chrome APIs: Utilizes Chrome's secure storage and permissions system
  • No Data Transmission: No analytics, tracking, or data collection servers
  • Open Source: Full transparency with publicly available source code

Third-Party Services

Quantum Tab may interact with the following third-party services only when explicitly configured by you:

  • GitHub API: For repository and pull request information
  • OpenAI API: For AI-generated background images (optional)
  • Google Gemini API: For AI-generated background images (optional)
  • Azure OpenAI API: For AI-generated background images (optional)

These services have their own privacy policies. We recommend reviewing them when enabling these integrations.

Your Rights and Control

You have complete control over your data in Quantum Tab:

  • Data Deletion: Remove the extension to delete all stored data
  • Widget Removal: Individual widgets clean up their data when removed
  • Export Capability: Access your data through Chrome's extension storage
  • API Key Management: Add, update, or remove third-party service connections
  • Feature Control: Enable or disable any feature at any time

Updates to This Policy

We may update this privacy policy to reflect changes in our extension or legal requirements. Any updates will be distributed through extension updates and will be effective immediately upon installation.
